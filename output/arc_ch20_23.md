## アーク概要

ガルド・レーヴ問題が本格的に動き出す章。
ルナが覚醒型センティエントウェポン「霜嵐弓・フローセン」と適合し、初めての戦闘を経験する。

---

## 章別プロット

### Ch20：修行と気づき
- ルナはヴァネッサへの報告後、「待つしかない」という状況の中で弓の修行を続ける
- シズクとの稽古（三連射の習得が目標）を複数回描く
- 章末：リーシャが旧校舎への2枚目の使用申請に気づき、ルナ（またはティア）に連絡

### Ch21：侵入日
- 4年生の調査実習のため、常勤教官がほぼ学園を離れている
- リーシャが申請を発見した時点で既に入館日当日
- 報告する時間も相手もない状況
- 鍵がなくなっていることを確認
- リーシャとシズクが旧実技室へ向かう
- ルナとティアがその「形相」を見て、声をかけずについていくことを決意
- 旧実技室の鍵が開いており、4人とも内部へ

### Ch22：管理区画
- リーシャ・シズクが管理区画でガルドと対峙・言い合い
- ルナとティアは咄嗟に身を隠す
- 隠れている最中、ルナが棚（または台）の上に弓を発見
- 惹かれるように手を伸ばし、触れた瞬間に淡く光る
- 光でバレる → 戦闘開始

### Ch23：戦闘・後処理
- リーシャ・シズクとルナ・ティアの4人でガルドと戦闘
- フローセンが霜刺・フリールをルナに伝える（武器が直接技を教える）
- ガルドを捕縛して終了
- 後処理：ルナ・ティアが無断で付いていったことを環主2人がかばう → 処分なし
- フローセンはルナが所持（本人に持たせる方が安全という判断）
- ただしセンティエントウェポンであることは周囲に言わないこととする

## 新登場人物：ガルド・レーヴ

| 項目 | 内容 |
|-----|------|
| 立場 | 非常勤講師（外部の人間） |
| 動機 | 管理区画に保管された古代魔導具・センティエントウェポン疑いの品を調査・盗み出す |
| 前歴 | 勤務外の日も学園に来ていた。ヴァネッサ教官は既に怪しんでいた |
| 捕縛後 | 捕縛される（逃走なし） |

### ヴァネッサ教官との関係
- 第19章でガルドの名前を告げた際、1〜2秒の沈黙があった
- これは「知らない人物」ではなく「すでに目をつけていた人物」だったため
- 勤務外に学園へ来る行動を把握しており、内心では警戒していた

## 新設定：霜嵐弓・フローセン

| 項目 | 内容 |
|-----|------|
| 形状 | 弓 |
| 属性 | 風・氷 |
| 原初の魔法 | 霜嵐 |
| 固定/覚醒 | 覚醒型 |
| 固有名 | フローセン |
| 発見場所 | 管理区画内（棚または台の上に保管されていた） |
| 外見 | 触れた瞬間に淡く光る（適合の証） |

### 覚醒型の特性（既存設定より）
- ルナの魔力を吸収して成長する（悪意ではなく元々の性質）
- 覚醒直後は魔力消費が激しい
- 外から中へ引き戻すイメージで制御する
- 武器自体が技を直接伝える

### 習得済みの技

#### 霜刺・フリール
- **習得タイミング**：Ch23の戦闘中（フローセンが直接伝える）
- **効果**：矢の着弾点から氷が広がり、敵の足元を凍らせて動きを封じる
- **用途**：足止め・動き制限（殺傷ではない）
- **frieren（独：凍る）由来の名**

## 管理区画に保管されていた品

- 学園の過去の教官記録・施設の改修履歴（ch19で判明済み）
- 古代魔導具（複数）
- センティエントウェポン疑いの品が一部含まれる（噂レベル）
- ガルドはこの噂を嗅ぎつけて調査していた
